# mindNote 项目结构

## 项目概述

mindNote 是一个智能情绪桌面摆件项目，通过环境感知和情绪日记记录，为用户提供情绪洞察和改善建议。

## MCP工具说明

### 1. 日记记录MCP (`diary-record-mcp`)

用于记录和管理情绪日记条目，支持情绪评分、标签、环境数据和用户备注的记录。

**主要功能：**
- 记录情绪日记条目
- 查询日记条目
- 更新现有条目
- 删除日记条目

### 2. 环境监测MCP (`emotion-monitor-mcp`)

用于监测环境条件，包括温度、湿度、光照水平和天气数据。

**主要功能：**
- 获取当前室内外环境数据
- 查询历史环境数据

### 3. 洞察生成MCP (`insight-generator-mcp`)

基于日记条目生成情绪洞察、趋势分析和改善建议。

**主要功能：**
- 生成情绪洞察
- 分析情绪趋势
- 导出日记数据
- 提供改进建议和积极鼓励

## 技术栈

- **后端**: Node.js + Express
- **协议**: MCP (Model Context Protocol)
- **数据格式**: JSON
- **语言**: JavaScript

## 使用方法

每个MCP工具都是独立的Node.js应用：

```bash
# 进入对应工具目录
cd mcp-tools/diary-record-mcp

# 安装依赖
npm install

# 启动服务
npm start
```

各工具端口分配：
- 日记记录MCP: 端口 3000
- 环境监测MCP: 端口 3001
- 洞察生成MCP: 端口 3002
