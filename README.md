# 智能情绪桌面摆件 (EmotionSense Desktop Companion)

一个基于MCP协议的智能桌面设备，能够感知环境、推断情绪并提供智能建议。

## 📖 项目简介

这是我的个人创新项目 - 一个智能情绪桌面摆件。通过环境传感器（温度、湿度、光照）采集数据，结合MCP (Model Context Protocol) 协议，让AI助手能够理解你的情绪状态并提供个性化建议。

## ✨ 核心特性

- 🌡️ **环境感知**: 实时监测温度、湿度、光照强度
- 🌤️ **天气集成**: 获取室外天气数据
- 🧠 **情绪推断**: 基于环境数据智能推断情绪状态
- 💡 **智能建议**: 提供个性化的环境优化和情绪调节建议
- 🔌 **MCP集成**: 标准MCP工具链，支持AI助手直接调用
- 📊 **数据洞察**: 长期追踪环境-情绪关联数据

## 🔧 技术架构

### 硬件方案
- **主控**: ESP32-S3 (WiFi + 蓝牙)
- **传感器**: SHT30 (温湿度) + BH1750 (光照)
- **显示**: 0.96寸 OLED
- **成本**: ~￥35-50

### MCP工具链
1. **environment-monitor-mcp** - 环境数据采集与查询
2. **emotion-inference-mcp** - 基于环境数据的情绪状态推断
3. **insight-generator-mcp** - 生成情绪洞察报告

### 数据流程
```
传感器 → ESP32 → 数据存储 → MCP工具 → AI助手 → 用户
   ↓        ↓         ↓         ↓        ↓       ↓
环境数据 → 本地处理 → 情绪推断 → 智能建议 → 改善环境
```

## 📊 项目特色

### 环境-情绪关联模型
基于科学数据建立环境对情绪的影响模型：
- 温度22-26℃，湿度40-60% → 舒适度高
- 结合天气状况调整判断（晴天加分，阴雨天减分）
- 个性化学习：根据用户反馈调整推断准确度

### 智能建议系统
- **环境优化**: "建议开窗通风5分钟" / "可以使用加湿器"
- **行为建议**: "适合进行轻度运动" / "可以听舒缓音乐"
- **生活提醒**: "记得多喝水" / "适当休息眼睛"

## 🚀 快速开始

### 硬件需求
- ESP32-S3 开发板 (~￥15)
- SHT30 温湿度传感器 (~￥8)
- BH1750 光照传感器 (~￥3)
- 0.96寸 OLED 显示屏 (~￥5)
- 杜邦线、面包板等

### 开发环境
- Arduino IDE / PlatformIO
- ESP-IDF v4.4+
- MCP SDK
- 天气API (和风天气/心知天气)

### 使用示例

**AI对话场景**:
```
用户: "我今天感觉有点闷"

AI:
我来查看你周围的环境数据：
- 室内湿度65%，有点偏高了
- 室外正在下雨，气压较低

这可能是你感觉闷的原因。建议：
1. 开窗通风5-10分钟
2. 喝杯温水
3. 我来播放一些轻松的音乐？
```

**MCP工具调用**:
```python
# 获取当前环境数据
env_data = get_current_environment()
# 返回: {temperature: 22.5, humidity: 65, weather: "sunny"}

# 推断情绪状态
mood = infer_current_mood(env_data)
# 返回: {comfort_level: 8, mood_state: "calm", energy_level: 7}

# 生成每日洞察
insight = generate_daily_insight("2025-12-29")
# 返回: 包含心情模式、环境影响、建议等
```
